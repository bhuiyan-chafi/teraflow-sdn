# Why you need this setup?

If you are doing multiple developments on teraflow and you need the official, development version running together! You need this setup. In this tutorial we are going to see how can we setup multiple `Ubuntu` instances using `vmware` and then running `tfs` in them.

## Install KVM

For this setup I am not going to use `vmware` or `virtualBox` because they are heavy. I found a better alternative `KVM` which is much mote lite than these solutions.
